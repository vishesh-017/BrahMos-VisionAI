# 🤝 Contributing to BrahMos VisionAI

Thank you for your interest in contributing! BrahMos VisionAI is an open-source AI surveillance platform and we welcome contributions of all kinds.

---

## 🚀 Getting Started

1. **Fork** the repository on GitHub
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/brahmos-visionai.git
   cd brahmos-visionai
   ```
3. **Create a branch** for your feature or fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```

---

## 📋 Contribution Guidelines

### Code Style
- **Python (Backend):** Follow [PEP 8](https://peps.python.org/pep-0008/). Use `black` for formatting.
- **TypeScript/React (Frontend):** Follow the existing ESLint config. Use functional components and React hooks.

### Commit Messages
Use clear, descriptive commit messages:
```
feat: add multi-camera stream switching
fix: resolve WebSocket reconnection issue
docs: update quick start guide
```

### Pull Requests
- Keep PRs focused on a single feature or bug fix
- Write a clear description of what the PR does
- Reference any related issues: `Closes #123`
- Ensure your code doesn't break existing functionality

---

## 🐛 Reporting Bugs

When reporting a bug, please include:
- Your OS and Python/Node.js version
- Steps to reproduce the issue
- Expected vs actual behavior
- Relevant logs or screenshots

Open a [GitHub Issue](https://github.com/vishesh-017/brahmos-visionai/issues) with the **bug** label.

---

## 💡 Suggesting Features

Feature requests are welcome! Open an issue with the **enhancement** label and describe:
- The problem you're trying to solve
- Your proposed solution
- Any alternatives you considered
